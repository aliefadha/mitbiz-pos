# AGENTS.md — mitbiz-pos

Monorepo with two npm workspaces: `frontend-pos` (React + Vite) and `backend-pos` (NestJS + Drizzle).

## Developer Commands

Run from repo root unless noted.

| Task | Command |
|------|---------|
| Install all deps | `npm install` |
| Lint/format both workspaces | `npm run lint` / `npm run format` |
| Check both workspaces | `npm run check` |
| Run frontend tests | `npm run test -w frontend-pos` |
| Run backend tests | `npm run test -w backend-pos` |
| Run backend e2e tests | `npm run test:e2e -w backend-pos` |

Pre-commit hook runs `npm run check -w backend-pos -w frontend-pos`. Commits must follow Conventional Commits (enforced by commitlint).

## Frontend (`frontend-pos/`)

- **Stack**: React 19, Vite, TanStack Router (file-based), TanStack Query, Tailwind CSS v4, shadcn/ui, Better Auth
- **Dev server**: `npm run dev` → http://localhost:3000
- **Build**: `npm run build` runs `vite build && tsc` (typecheck happens after build, not during)
- **Test**: Vitest (`npm run test`)
- **Lint/Format**: Biome v2. `routeTree.gen.ts` is ignored in `biome.json`
- **Router**: File-based routes in `src/routes/`. `routeTree.gen.ts` is auto-generated by the TanStack Router Vite plugin—do not edit manually
- **Strict TS**: `noUnusedLocals` and `noUnusedParameters` are enabled
- **Alias**: `@/` maps to `src/`

## Backend (`backend-pos/`)

- **Stack**: NestJS, Drizzle ORM, PostgreSQL, Better Auth
- **Dev server**: `npm run start:dev` → port 3001 (or `PORT` env var)
- **Build**: `npm run build` runs `biome check && nest build`
- **Test**: Vitest (`npm run test`). E2E tests use `test/jest-e2e.json`
- **Lint/Format**: Biome v1 (different version from frontend)
- **Database**:
  - Requires PostgreSQL. Local dev uses Docker Compose (`docker-compose up -d` in `backend-pos/`) which starts Postgres on **5435** and Mailpit on **8025**
  - `DATABASE_URL` must be set (see `.env.example`)
  - Schema lives in `src/db/schema/`
  - Commands: `db:generate`, `db:push`, `db:migrate`, `db:seed`, `db:studio`
- **Alias**: `@/` maps to `src/`
- **CORS**: Origins read from comma-separated `ALLOWED_ORIGINS` env var
- **Static files**: Uploads served from `/uploads/` path. `bodyParser` is disabled in `main.ts` (important for multer/file uploads)
- **API docs**: Swagger UI at `/api/docs`, Scalar at `/api/reference`

## Style & Conventions

- Both workspaces use Biome, **not** ESLint/Prettier.
- Frontend and backend use different Biome versions and slightly different rule sets—do not assume configs are identical.
- Prefer single quotes, 2-space indent, 100 char line width (enforced by Biome).
